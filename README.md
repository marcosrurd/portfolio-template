# Mi Portfolio
## AQUÍ PUEDES ENCONTRAR LA WEB. https://marcosrurd.github.io/portfolio-template/ 

### El proyecto consiste en el diseño y desarrollo de mi portfolio web personal, con el objetivo de presentar mi perfil como diseñador grafico de una forma profesional y visual.
### El portfolio refleja mis intereses principales, especialmente el diseño editorial, el ámbito audiovisual y la dirección creativa y de arte.
## Estructura de la web:
### La web está planteada como una sola página organizada en disitintas secciones como:
### Hero section/introducción: Pantalla inicial con animación de entrada que introduce la identidad del portfolio. Además, cuenta con una barra de scroll de algunos de mis proyectos.
### About me: Presentación personal y enfoque profesional.
### Projects: Selección de proyectos mostrados mediante tarjetas interactivas.
### Contact: Información de contacto y enlaces a redes sociales.
### Además, en el header se encuentran los apartados de la web y cuentan con un botón para cambiar la web a modo oscuro.
